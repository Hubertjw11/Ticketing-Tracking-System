**Ticketing Tracking System** adalah aplikasi berbasis web yang dirancang untuk mempermudah pengelolaan, pelacakan, dan penyelesaian laporan masalah atau permintaan layanan (tickets) secara efisien.

- **Pelacakan Status:** Memantau status tiket secara real-time (misal: Open, In Progress, Resolved, Closed).
- **Dashboard Admin:** Panel khusus untuk admin/teknisi mengelola dan menanggapi tiket yang masuk.
- **Notifikasi:** Pemberitahuan otomatis saat ada pembaruan pada tiket (opsional).
- **Riwayat Tiket:** Rekam jejak lengkap dari setiap tiket yang pernah dibuat.
